# 🎬 YouTube Homepage Clone

A responsive **YouTube homepage clone** built using **HTML and CSS**.
This project replicates the layout and basic UI of the YouTube home page including the **header, sidebar, video grid, and shorts section**.

The goal of this project was to practice **modern CSS layout techniques** such as **Flexbox, CSS Grid, responsive design, and hover animations**.

🚀 Features

* 📺 Responsive **video grid layout**
* 🎬 **YouTube Shorts** horizontal scrolling section
* 📌 **Fixed header navigation bar**
* 📂 **Fixed sidebar navigation**
* 🔍 Search bar with icon
* 🔔 Notification icon with badge
* 🎨 Dark theme similar to YouTube
* ✨ Hover animations for videos and buttons
* 📱 Basic responsive design using media queries

## 🛠️ Built With

* **HTML5**
* **CSS3**
* **Flexbox**
* **CSS Grid**
* **Media Queries**
* **Google Fonts (Roboto)**
